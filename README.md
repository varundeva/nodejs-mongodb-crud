**Clone Project or Download zip and Extract**
```sh
https://github.com/varundeva/nodejs-mongodb-crud.git
```
**Install Required npm Packages**
```sh
$ cd nodejs-mongodb-crud
$ npm i express
$ npm i mongodb
$ npm i body-parser
```
**Run Node Project**
```sh
-app.js is entry point
-run node app.js
-using Post Man Software interact
```
